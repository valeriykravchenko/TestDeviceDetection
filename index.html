Browser Detection Test

This is a simple test to understand what libraries can give us neseccary information about mobile browsers.

As a result of this test managed to see that none of the libraries give us full information of device, specifically it doesn't provide us information about device type. As a solution we can use simple method: to detect device width and height, detect smallest value and get it as a device formfactor (phone, tablet or desktop).
